How You Can Help

Reporting Issues #

If you find a bug, mistake, or an inconsistency in the Odin code or documentation, please let us know by filing a ticket on our issue tracker.

Contributing Code #

Check out the Odin source code.

Source Code Odin is an open source project and we welcome contributions from the community.

Check the issue tracker for open issues that may interest you. Those labelled help wanted are particularly in need of help from the outside.
